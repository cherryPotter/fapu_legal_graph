## 案件事实：

被告人任某某。原西安市临潼区文化和旅游体育局人民体育场场长。因涉嫌贪污罪，于2023年10月20日被西安市临潼区监察委员会采取留置措施。2024年1月29日经西安市临潼区人民检察院决定，同日由西安市公安局临潼分局执行拘留，同年2月7日经西安市临潼区人民检察院决定逮捕，同日由西安市公安局临潼分局执行逮捕。现羁押于西安市临潼区看守所。
　　辩护人赵继强，陕西萃聚律师事务所律师。
　　西安市临潼区人民检察院以临潼检刑诉〔2024〕Z1号起诉书指控被告人任某某犯贪污罪，向本院提起公诉。本院受理后，依法组成合议庭，适用认罪认罚普通程序公开开庭审理了本案。西安市临潼区人民检察院指派检察员程阳出庭支持公诉，被告人任某某及其辩护人赵继强到庭参加诉讼。现已审理终结。
　　公诉机关指控，
　　贪污罪
　　1.2014年3月至2015年6月，被告人任某某在担任西安市临潼区文化体育广播电视局体育场场长期间，利用职务上的便利，与韩某挂靠的西安市临潼区建筑工程有限公司分别签订了临潼区体育场400ｍ跑道维修及排水沟维修工程施工合同，套取工程款9万元。任某某将其中的3万元陆续全部用于购买彩票牟利。
　　2.2016年3月至12月，被告人任某某利用担任临潼区体育场场长的职务便利，与韩某的西安市兴博置业有限公司分别签订了临潼区体育场国民体质测试与科学健身指导站装修、办公楼幕墙维修及办公楼屋面维修工程施工合同，套取工程款5万元。任某某将其中的3万元借给王顺利(已死亡)使用。
　　3.2016年年底至2017年4月，被告人任某某利用担任临潼区体育场场长的职务便利，与韩某的西安市兴博置业有限公司签订了名为临潼区体育场乒乓球场地、周边环境刷新扮靓工程的虚假施工合同，套取工程款12万元。任某某将其中的3万元用于其女出阁宴购买烟酒。
　　4.2018年3月至8月，被告人任某某利用担任临潼区体育场场长的职务便利，与韩某的西安恒邦建筑工程有限公司签订了临潼区体育场北侧外环路面维修改造工程施工合同，套取工程款3万元。任某某将其中的2万元用于其父生日宴购买烟酒。
　　5.2019年5月至6月，被告人任某某利用担任临潼区体育场场长的职务便利，通过编造教练员费用套取单位专项资金6万元。任某某将其中的5万元用于其外孙满月宴购买烟酒。
　　6.2019年7月至12月，被告人任某某利用担任临潼区体育场场长的职务便利，与孙某的西安恰豪建筑装饰工程有限公司分别签订了临潼区体育场兵乓球室翻新维修、室外路面硬化及零星工程施工合同，套取工程款6万元。任某某将其中的3万元用于房屋装修。
　　受贿罪
　　1.2017年3月，被告人任某某在担任西安市临潼区文化和旅游体育局人民体育场场长期间，利用职务上的便利，为张某2在体育场田径场改造提升项目监理费支付中提供帮助，并收取张某22万元现金。
　　2.2017年5月，被告人任某某利用担任临潼区体育场场长的职务便利，为杨某1在其承揽体育场田径场改造提升工程沥青混凝土路面铺筑工程中提供帮助，并收取杨某12万元现金。
　　3.2017年7月，被告人任某某利用担任临潼区体育场场长的职务便利，为姜某在其承揽体育场田径场改造提升工程塑胶跑道和人造草坪铺设业务中提供帮助，并收取姜某1万元现金。
　　4.2016年11月至12月期间，被告人任某某利用担任临潼区体育场场长的职务便利，以让王某1承包体育场水电费未收取好处和自己急需用钱为由，向王某1索要7万元好处费。
　　5.2019年7月至2023年5月期间，被告人任某某利用担任临潼区体育场场长的职务便利，以其帮助王某1承揽工程为由，以0.1万元至2万元不等的金额，通过微信转账方式先后向王某1索要29笔共计14.9万元好处费。
　　被告人任某某利用职务上的便利，索要及非法收受他人财物共计26.9万元。任某某将其中的5万元用于房屋装修，9.622万元通过微信转给张某3，7万元借给王顺利，剩余5.278万元用于个人日常消费。
　　公诉机关提交了案件线索来源及到案经过；立案决定书、留置决定书、延长留置时间决定书；被告人任某某的供述和辩解；证人李某、韩某、张某1、孙某、张某2、杨某1、姜某、王某1、王某2、郭某、高某、杨某2、何某、马某、张某3等人的证言；体育场专项资金、补助资金文件通知、拨款表等；建设施工合同、建设工程中标通知书、建设工程监理合同、工程竣工验收报告、水电承包合同等；体育场会计凭证、银行账户明细单、“小金库”账务资料等；微信支付交易明细；任职通知、职务任免文件、工资变动档案表、临潼区体育局编制文件及领导分工文件；户籍证明、无犯罪记录证明等证据证实。为证实上述指控，公诉机关当庭宣读、出示了上述证据。
　　公诉机关认为，被告人任某某身为国家工作人员，利用职务上的便利，非法占有公共财物，数额较大；身为国家工作人员，利用职务上的便利，索取他人财物，非法收受他人财物，为他人谋取利益，数额巨大，其行为已触犯《中华人民共和国刑法》第三百八十二条第一款、第三百八十五条第一款之规定，应当以贪污罪、受贿罪追究其刑事责任。被告人到案后自愿如实供述自己的犯罪事实，自愿认罪认罚，且家属代替其退赃2万元依法可以从轻处罚。建议以贪污罪判处被告人有期徒刑二年六个月，并处罚金；以受贿罪判处被告人任某某有期徒刑三年，并处罚金。
　　被告人任某某对指控事实、罪名及量刑建议没有异议且签字具结，在开庭审理过程中亦无异议。辩护人认为，本案部分事实不清，证据不足。从客观方面讲，认定任某某贪污的客观证据不足，从主观方面讲，任某某贪污的主观故意不明显。对于任某某受贿罪，其认为第四宗受贿的证据不足，第五宗受贿罪事实不清。综上，依照法律规定和任某某的犯罪事实，任某某不构成贪污罪，任某某的受贿罪应认定为5万元，量刑应为三年以下有期徒刑，建议对任某某判处缓刑。

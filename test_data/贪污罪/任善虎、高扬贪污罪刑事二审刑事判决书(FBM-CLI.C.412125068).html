任善虎、高扬贪污罪刑事二审刑事判决书

山东省德州市中级人民法院
刑事判决书
(2022)鲁14刑终14号

　　公诉机关抗诉机关山东省平原县人民检察院。
当事人　　原审被告人任善虎。2011年7月任平原县住房和城乡建设局(以下简称平原县住建局)科员，2015年8月任平原县住建局拆迁办副主任，2019年6月任平原县住建局城市建设发展中心房屋征收股副股长，2021年9月6日被开除党籍、公职。因涉嫌严重违纪违法，于2021年8月12日被平原县监察委员会留置；因涉嫌犯贪污罪，于2021年9月7日被刑事拘留；于2021年9月20日被执行逮捕。现羁押于德州市看守所。
　　辩护人张立峰、张佳欣(实习)，北京市大嘉(德州)律师事务所律师。
　　原审被告人高扬。因涉嫌严重违法，于2021年8月12日被平原县监察委员会留置；因涉嫌犯贪污罪，于2021年9月7日被刑事拘留；于2021年9月20日被执行逮捕。2021年12月22日被平原县人民法院取保候审。
　　辩护人张海霞，山东指南针律师事务所律师。
审理经过　　山东省平原县人民法院审理山东省平原县人民检察院指控原审被告人任善虎、高扬犯贪污罪一案，于2021年12月22日作出(2021)鲁1426刑初183号刑事判决。宣判后，山东省平原县人民检察院提出抗诉。本院受理后，依法组成合议庭，于2022年2月23日公开开庭进行了审理。德州市人民检察院指派检察员石青、检察官助理王吉超出庭履行职务，原审被告人任善虎及其辩护人张立峰、张佳欣(实习)、原审被告人高扬及其辩护人张海霞到庭参加诉讼。现已审理终结。
一审法院查明　　原审判决认定：
　　2020年6月至2021年2月，被告人任善虎与被告人高扬商议，利用任善虎担任平原县住建局城市建设发展中心房屋征收股副股长负责给拆迁安置户出具安置楼房结算单、安置居住权凭证的职务便利，采取虚构拆迁安置户的方式，将三套拆迁安置楼房套出，并由被告人高扬卖掉。经评估，被告人任善虎、高扬擅自出售的三套拆迁安置房在出售日的市场价值为185.7万元。具体事实如下：
　　1.2020年6月，被告人任善虎、高扬擅自将南城社区10号楼2单元602室出售给田某，获得售房款53.5万元。
　　2.2020年11月，被告人任善虎、高扬擅自将南城社区18号楼2单元501室出售给侯某，获得售房款59万元。
　　3.2021年2月，被告人任善虎、高扬擅自将贺家行社区14号楼2单元804室出售给李某，获得售房款54万元。
　　以上楼房售房款共计166.5万元，其中任善虎实际分得119万元，高扬实际分得43.2959万元，剩余4.2041万元由高扬支付了上述楼房的住房维修资金和因卖房而产生的中介费。
　　经查，被告人任善虎已上缴违法所得28万元；被告人高扬已上缴违法所得55.9775万元。
　　上述事实，有下列经原审庭审质证的汽车等物证；户籍证明、任职证明、发破案经过、安置楼房结算单、安置居住权凭证、银行交易明细、房地产估价报告书等书证；田某、侯某、李某等证人证言；被告人任善虎、高扬的供述和辩解等证据予以证实。
一审法院认为　　原审法院认为，被告人任善虎身为国家工作人员，利用职务上的便利，与被告人高扬共同非法占有公共财物，数额巨大，二被告人的行为已构成贪污罪。鉴于被告人任善虎、高扬到案后如实供述其犯罪事实，属坦白，且自愿认罪认罚，依法可予以从轻处罚；被告人高扬在共同犯罪中起次要作用，属从犯，应当从轻处罚；被告人任善虎、高扬已上缴部分违法所得，可酌情从轻处罚。依照《中华人民共和国刑法》第三百八十二条第一款、第三款、第三百八十三条第一款第(二)项、第二款、第三款、第二十五条第一款，第二十七条、第六十七条第三款、第六十二条、第六十四条、第七十二条第一款、第七十三条第二款、第三款、第五十二条、第五十三条、《中华人民共和国监察法》第四十四条第三款、《最高人民法院最高人民检察院关于办理贪污贿赂刑事案件适用法律若干问题的解释》第二条第一款、第十八条、第十九条第一款及《中华人民共和国刑事诉讼法》第十五条之规定，以被告人任善虎犯贪污罪，判处有期徒刑四年五个月,并处罚金人民币二十五万元。以被告人高扬犯贪污罪，判处有期徒刑三年，缓刑五年，并处罚金人民币二十万元。被告人任善虎、高扬贪污共计人民币一百八十五万七千元应予以没收，上缴国库(其中已追缴二被告人人民币共计八十三万九千七百七十五元；依法扣押的被告人任善虎贪污款所购鲁ＮＷ××××号大众探岳汽车一辆、古铜钱一宗依法变现，变现后依法上缴国库，不足部分继续追缴)。
　　宣判后，山东省平原县人民检察院提出抗诉。
　　山东省平原县人民检察院的抗诉理由：1.一审判决对被告人高扬适用缓刑，系适用法律错误。被告人高扬与任善虎三次擅自出售三套拆迁安置房，市场价值185.7万元，贪污数额巨大，二人系共同犯罪，应当共同承担责任。二被告人的犯罪行为和后果，社会影响较大，被告人高扬退赃数额虽然多于本人实际所得数额，但是并未全部退赃，不应当认定为《中华人民共和国刑法》第七十二条第一款第一项规定的“犯罪情节较轻”。2.一审判决书未对“平原县监察委员会冻结任善虎建设银行账户(6227××××2910)1502.96元、冻结任善虎父亲任某农业银行账户(6228××××3761)24612.2元、查封任善虎及高扬房产”做出处理决定，违反了《中华人民共和国刑事诉讼法》第二百四十五条第三款，系适用法律错误。
　　德州市人民检察院支持刑事抗诉意见书认为抗诉正确，应予支持。另提出，一审判决认定高扬为从犯，系事实认定错误。虽然出售拆迁安置房这一行为必须依靠任善虎负责房屋拆迁安置工作的职务便利，但出售拆迁安置房的过程中，是由高扬全程负责寻找买家、签订合同并收取房款，高扬在整个过程中是积极参与者，所起作用较大，不应认定为从犯。
　　出庭检察员的出庭意见：1.一审判决认定事实错误。高扬积极参与并全程负责寻找买家、签订合同并收取房款，其行为同样起到非常大的作用，不应认定为从犯。2.案件诉讼程序合法。3.一审判决适用法律错误。(1)一审判决对高扬适用缓刑，系适用法律错误。(2)一审判决书未对“平原县监察委员会冻结任善虎建设银行账户(6227××××2910)1502.96元、冻结任善虎父亲任某农业银行账户(6228××××3761)24612.2元、查封任善虎及高扬房产”做出处理决定，违反了《中华人民共和国刑事诉讼法》第二百四十五条第三款，系适用法律错误。综上，一审判决认定事实错误，适用法律错误，平原县人民检察院抗诉理由正确，应予支持，建议二审法院对高扬撤销缓刑，并对涉案财物依法判决。
　　原审被告人任善虎、高扬均无辩解。
　　原审被告人任善虎的辩护人的辩护意见：1.对一审认定原审被告人任善虎的罪名及量刑没有异议，任善虎真心认罪、悔罪,且任善虎构成坦白、认罪认罚，上缴部分违法所得，可以从轻处罚。2.任善虎的账户资金属于其工资，任善虎愿意听从法院处理，其父亲账户上的资金，因属于赃款，可以依法追缴。3.望人民法院解除对其房产的查封。被告人任善虎名下的房产，属于在案发前用自己的积蓄和从亲属处的借款支付首付款而购买取得，贷款也均是用任善虎的工资偿还，属于任善虎夫妻的共同财产，与本案没有关联性，任善虎的妻子和两个年幼的孩子仅有这一套房屋用于居住，因此，本案中查封的房屋属于被告人的合法财产，且涉及他人利益，不属于违法所得，不能进行追缴。
　　原审被告人诉称原审被告人高扬的辩护人的辩护意见：在刑事抗诉书认可对高扬判处三年有期徒刑的情况下，高扬完全符合缓刑的适用条件。具体理由：1.高扬作为从犯、无公职人员身份，经过社区调查宣告缓刑对所居住社区无重大不良影响，有坦白、认罪认罚、积极退赃、缴纳罚金、主观恶性较小、真心悔改、人身危险性较小、系初犯、偶犯等法定、酌定从轻减轻处罚情节。2.人民检察院抗诉称对高扬宣告缓刑系适用法律错误无法律依据，高扬不具有法律规定的一般不适用缓刑的情节，不具有最高人民法院、最高人民检察院《关于办理职务犯罪案件严格适用缓刑、免予刑事处罚若干问题的意见》第二条规定的一般不适用缓刑或者免予刑事处罚的情形。3.平原县人民检察院起诉书认定高扬是从犯，抗诉书也未针对一审判决认定高扬为从犯提起抗诉，市检察院认为高扬不是从犯从程序上违法，违反了有利于被告人的原则。且高扬不是犯意的提出者，得赃数额少，作用相对较小，属于从犯。原审被告人高扬作为从犯，将其违法所得退赔应当视为全额退赃退赔，抗诉书称“二被告人的犯罪行为和后果，社会影响较大，应当共同承担责任”的意见不符合刑事案件退赃退赔的精神，让作为从犯的高扬承担全案的赃款退赔不符合罪责相适应原则。4.原审被告人高扬具有自首情节，适用缓刑并无不当。2021年8月11日平原县住建局的冯立远电话通知高扬到住建局，称局长要和高扬谈话，此时高扬已经知道事情被发现，但仍然去了平原县住建局，到了后冯立远称纪委监委要来和高扬谈，高扬在现场等待，纪委来人后，将高扬带走，高扬到案后如实供述了自己关于南城社区10号楼2单元602室的犯罪事实。根据《最高人民法院关于处理自首和立功若干具体问题的意见》中关于明知他人报案而在现场等待，抓捕时无拒捕行为，供认犯罪事实的，应当认定为自首的规定，本案中高扬关于南城社区10号楼2单元602室的犯罪行为应当认定为自首。
本院查明　　经二审审理查明的事实及证据与一审一致。
　　针对平原县人民检察院的抗诉意见、德州市人民检察院支持刑事抗诉意见、出庭检察员的出庭意见、原审被告人任善虎和高扬的辩护人的辩护意见，根据本案事实和证据，依照法律，本院综合评判如下：
　　关于出庭检察员提出“原审被告人高扬不属从犯”的出庭意见，及高扬的辩护人提出“高扬系从犯”的辩护意见。经查，原审被告人任善虎系国家工作人员，利用其职务便利，与原审被告人高扬共谋，虚构拆迁安置户套取三套房产，由高扬销售，销售赃款三七分成。任善虎与高扬系共同犯罪，利用的是任善虎的职务便利，高扬负责将房屋销售，起到的是次要、辅助作用，且高扬只分得三成赃款，即使第三套房屋是高扬提出的套取，也要得到任善虎同意并实施才能完成，因此高扬在共同犯罪中属于从犯。抗诉机关在指控该案的起诉书中亦明确写明高扬起次要作用，符合刑法第27条的规定，一审判决据此认定高扬系从犯并无不当。德州市人民检察院支持刑事抗诉意见书及出庭检察员意见认为一审认定高扬从犯不当的意见不能成立，且也超出了抗诉范围。因此出庭检察员的此项意见不能成立，本院不予采纳，原审被告人高扬的辩护人的此项辩护意见成立，本院予以采纳。
　　关于抗诉机关及出庭检察员提出“对原审被告人高扬适用缓刑属于适用法律错误”的抗诉理由及出庭意见，高扬的辩护人提出“对高扬适用缓刑并无不当”的辩护意见。经查，虽然原审被告人任善虎与高扬共同贪污数额巨大，但高扬在共同犯罪中系从犯，主动到案，如实供述了小部分犯罪事实，系坦白，认罪认罚，认罪悔罪态度较好，其个人违法所得赃款已全部退缴，并多退2000余元，主动缴纳罚金。虽然公诉机关与高扬签署认罪认罚决定书时签署的量刑意见是有期徒刑三年，但一审庭审时对于高扬是否可适用缓刑问题，一审法院征求了公诉机关的意见，公诉机关表示由法院依法决定。一审法院对高扬适用缓刑由社区矫正机构进行了评估调查，经调查认为对高扬判处缓刑对所居住社区没有重大不良影响。故原审判决对高扬判处缓刑并无不当。因此抗诉机关的上述抗诉意见、出庭检察员的上述出庭意见均不能成立，本院不予采纳；原审被告人高扬的辩护人的此项辩护意见成立，本院予以采纳。
　　关于任善虎的辩护人提出“任善虎真心认罪、悔罪,且任善虎构成坦白、认罪认罚，上缴部分违法所得，可以从轻处罚”的辩护意见。经查，原审判决对其坦白、认罪认罚、上缴部分违法所得的从轻情节已考虑，并已从轻处罚，量刑并无不当。因此上述辩护意见不能成立，本院不予采纳。
　　关于高扬的辩护人提出“对高扬主动到案如实供述的南城社区10号楼2单元602室的犯罪行为应认定为自首”的辩护意见。经查，原审被告人高扬经传唤主动到案，但其只供述了套取南城社区10号楼2单元602室的犯罪行为，没有如实供述主要罪行，不能认定为自首。故上述辩护意见不能成立，本院不予采纳。
　　关于抗诉机关及出庭检察员提出“一审判决对平原县监察委员会冻结任善虎建设银行账户(6227××××2910)1502.96元、冻结任善虎父亲任某农业银行账户(6228××××3761)24612.2元、查封任善虎及高扬房产未做出处理决定，属适用法律错误”的抗诉意见和出庭意见，及任善虎的辩护人提出“任善虎的账户资金属于其工资，愿意听从法院处理，其父亲账户上的资金，因属于赃款，可以依法追缴;任善虎名下的房产，属于在案发前用合法资金购买取得，贷款也均是用任善虎的工资偿还，属于任善虎夫妻的共同财产，与本案没有关联性，任善虎的妻子和两个年幼的孩子仅有这一套房屋用于居住，因此，本案中查封的房屋属于被告人的合法财产，且涉及他人利益，不属于违法所得，不能进行追缴”、高扬的辩护人提出“高扬作为从犯，将其违法所得退赔应当视为全额退赃退赔，让作为从犯的高扬承担全案的赃款退赔不符合罪责相适应原则”的辩护意见。经查，根据《最高人民法院关于适用<中华人民共和国刑事诉讼法>的解释》第四百四十九条第二款的规定，查封、扣押、冻结的财物属于被告人合法所有的，应当在赔偿被害人损失、执行财产刑后及时返还被告人。任善虎案发后未全部退赃，其建设银行尾号2910账户中的资金1502.96元系其合法资产，任善虎父亲任某名下农业银行尾号3761账户一直由任善虎使用，其中的资金系赃款。因此对查封的上述账户中的资金应予没收，抵顶赃款。查封的任善虎房产，系其夫妻共同财产，其家庭只有该一套房产，该房产是案发前购买，没有证据证明用赃款还过贷款，但任善虎未全部退赃，应依法继续追缴其赃款。查封的高扬的房产，也系其夫妻共同财产，其家庭只有该一套房产，高扬作为从犯，已将其犯罪所得全部退缴，不宜再从高扬处追缴赃款。因此上述抗诉意见及出庭检察员意见部分成立，本院部分予以采纳。高扬的辩护人的上述辩护意见成立，本院予以采纳。任善虎的辩护人的上述辩护意见部分成立，本院部分予以采纳。
本院认为　　本院认为，原审被告人任善虎身为国家工作人员，利用职务上的便利，与原审被告人高扬共同非法占有公共财物，数额巨大，其行为均构成贪污罪。抗诉机关的抗诉意见部分成立，本院部分予以支持，原审被告人任善虎、高扬的辩护人的辩护意见部分成立，本院部分予以采纳。原审判决认定事实清楚，证据确实、充分，定罪准确，量刑适当，审判程序合法，但对部分涉案财物未作出处理决定不当。据此，依照《中华人民共和国刑法》第三百八十二条第一款、第三款、第三百八十三条第一款第(二)项、第二款、第三款、第二十五条第一款、第二十六条、第二十七条、第六十七条第三款、第六十一条、第六十二条、第六十四条、第七十二条、第七十三条、第五十二条、第五十三条、《中华人民共和国监察法》第四十四条第三款、《最高人民法院、最高人民检察院关于办理贪污贿赂刑事案件适用法律若干问题的解释》第二条第一款、第十八条、第十九条第一款及《中华人民共和国刑事诉讼法》第十五条、第二百四十五条第三款、第二百三十六条第一款第(二)项、《最高人民法院关于适用<中华人民共和国刑事诉讼法>的解释》第四百四十九条第二款之规定，判决如下：
裁判结果　　一、维持山东省平原县人民法院(2021)鲁1426刑初183号刑事判决对原审被告人任善虎、高扬的定罪量刑部分，即“被告人任善虎犯贪污罪，判处有期徒刑四年五个月，并处罚金人民币二十五万元。被告人高扬犯贪污罪，判处有期徒刑三年，缓刑五年，并处罚金人民币二十万元”。
　　二、撤销山东省平原县人民法院(2021)鲁1426刑初183号刑事判决对涉案财产的判决部分，即“被告人任善虎、高扬贪污共计人民币一百八十五万七千元应予以没收，上缴国库(其中已追缴二被告人人民币共计八十三万九千七百七十五元；依法扣押移送本院的被告人任善虎贪污款所购鲁ＮＷ××××号大众探岳汽车一辆、古铜钱一宗依法变现，变现后依法上缴国库，不足部分继续追缴)”。
　　三、原审被告人任善虎退缴的违法所得赃款人民币二十八万元、原审被告人高扬退缴的违法所得赃款人民币共计五十五万九千七百七十五元予以没收，上缴国库；冻结的任善虎建设银行尾号2910账户人民币1502.96元、冻结的任善虎的父亲任某农业银行尾号3761账户人民币24612.2元抵顶赃款，予以没收，上缴国库；依法扣押的任善虎用赃款购买的登记在其妻付聪名下的鲁ＮＷ××××号大众探岳汽车一辆及古铜钱一宗依法变现，变现款抵顶赃款予以没收，上缴国库；不足部分继续向任善虎追缴。
　　本判决为终审判决。
